setting up the API
To setup sayuki, all you need to do is the following:

Step 1: Set your proxy URL / endpoint URL to https://sayuki-proxy.com/v1/chat/completions — or for ST/CHUB/TAVO users, https://sayuki-proxy.com/v1

Step 1.5: On ST/CHUB, you'll need to set it to be an OpenAI Reverse Proxy or anything similar

Step 2: Get a model name from the Models page, accessible from the main page

Step 3: Set your API key. Create one in the Keys section on the main page — it will be attached to the provider of your choice

Step 4: Profit
